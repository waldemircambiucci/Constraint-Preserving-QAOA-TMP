# Constraint-Preserving-QAOA-TMP
Constraint-Preserving Quantum Approximate Optimization for the Train Marshalling Problem

The main goal for this project is:
Position TMP as a high-impact railway logistics problem with strong potential for hybrid quantum optimization.
Introduce a TMP-specific constrained QAOA approach tailored to operational railway constraints with the Railway-Constrained Mixer (RCM) to preserve feasibility during optimization.
Highlight the broader potential of Variational Quantum Algorithm (VQA) and domain-structured quantum optimization for port-related logistics operations.
